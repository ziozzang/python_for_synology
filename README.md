python_for_synology
===================

Python ENV for Synology NAS. You can install pip for synology.

Prerequsit
 - Python Package installed. use Package Manager on GUI Menu.

```
curl https://raw.github.com/ziozzang/python_for_synology/master/install-pip | sh
```
